RGB (Red, Green, Blue)
RGB is a color model where each color is represented by a combination of three primary colors: red (R), green (G), and blue (B). Each of these components can have values ranging from 0 to 255. The combination of these three colors in different proportions allows for a wide range of colors. For example, full intensity of red, without green and blue, gives us the color red (RGB 255, 0, 0).

CMYK (Cyan, Magenta, Yellow, Key/Black)
CMYK is a color model primarily used in printing. It consists of four primary colors: cyan (C), magenta (M), yellow (Y), and black (K). Unlike the RGB model, in CMYK, color is achieved by subtracting colors from white light. Each component in CMYK represents the percentage of coverage of a given area on paper by that color. For example, full intensity of the red hue is achieved by not covering the area with cyan, magenta, and yellow (CMYK 0%, 100%, 100%, 0%).
